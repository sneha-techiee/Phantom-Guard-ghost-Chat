# Phantom Guard

**Ghost account simulation system for detecting fraud patterns**

Phantom Guard is a real-time AI-driven system designed to **simulate ghost accounts** and **detect potentially fraudulent interactions**. It helps users understand suspicious behaviors online and educates them about safe communication practices.
